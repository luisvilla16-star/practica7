# Práctica 7: Herencia en Java – Sistema de Monitoreo de Sensores

## Versión completa

### Descripción
Esta práctica consiste en implementar un **Sistema de Monitoreo de Hardware Distribuido** utilizando **herencia, clases abstractas y polimorfismo** en Java.  
El sistema incluye:

- Una clase base `DispositivosElectronicos`.
- Una clase abstracta `Sensor`.
- Subclases concretas de sensores: `SensorTemperatura`, `SensorPresion` y `SensorAceleracion`.
- Una clase principal `SistemaMonitoreo` para probar el funcionamiento de los sensores.

Cada sensor genera valores simulados aleatorios y permite probar encendido, calibración y lectura de datos.
